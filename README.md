This repository contains an application that generates ready-to-use PDF templates for theses at Halmstad University. The goal is to simplify the creation of title pages, covers, and basic document structure so that students can quickly start working with a correctly formatted report.
Features
* Generates PDF templates for theses
* The desired visual identity is added as PDF files in the PDF library.
* Supports multiple levels/types of theses (e.g., bachelor, master, and advanced level)
